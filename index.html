JOGO DO NÚMERO SECRETO
[ JAVASCRIPT ]
[ CSS ]

No Número Secreto, vamos estudar, analisar, planejar e codar em como iríamos criar uma função ao qual o usuário teria que escolher um número de 0 a 10.

Caso o usuário não insira nada e tente chutar, a página irá dar uma janela de alerta dizendo para preencher o campo corretamente. E se tentar colocar um número menor que 0 e maior que 10, também aparecerá uma mensagem de erro, pedindo para preencher o campo corretamente.

Além disso, após inserir o número e clicar em chutar, a página vai mostrar uma mensagem dizendo se o número escolhido é maior ou menor que o Número Secreto até que o usuário consiga acertar. O botão de Novo jogo vai estar disponível para reiniciar o campo e assim reiniciar o jogo.
